FMod

Divides two floating-point numbers and returns the floating-point remainder. This differs from VBScript's Mod operator, which divides two integers and returns the integer remainder.

Syntax

Rhino.FMod (dblX, dblY)

Parameters

dblX

Required.  Number.  The first value.

dblY

Required.  Number.  The second value.

Returns

Number

The floating-point remainder of x / y if successful.

Null

On error.

Example

MsgBox Rhino.FMod (-10.0, 3.0)
