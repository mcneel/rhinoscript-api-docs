LightCount

Returns the number of light objects in the document.

Syntax

Rhino.LightCount ()

Parameters

None.

Returns

Number

The number of lights in the document.

Example

Dim intCount

intCount = Rhino.LightCount

Rhino.Print "There are " & CStr(intCount) & " lights."
