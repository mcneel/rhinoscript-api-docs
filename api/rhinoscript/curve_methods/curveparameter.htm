CurveParameter

Converts a normalized curve parameter to a curve parameter; one within the curve's domain.

Syntax

Rhino.CurveParameter (strObject , dblParameter)

Parameters

strObject

Required.  String.  The object's identifier.

dblParameter

Required.  Number.  The normalized curve parameter to convert.

Returns

Number

The curve parameter if successful.

Null

On error.

Example

Dim strObject, dblParameter, dblNormalized

strObject = Rhino.GetObject("Select curve")

If Rhino.IsCurve(strObject) Then

  dblNormalized = 0.5

  Rhino.Print "Normalized parameter: " & CStr(dblNormalized)

  dblParameter = Rhino.CurveParameter(strObject, dblNormalized)

  Rhino.Print "Curve parameter: " & CStr(dblParameter)

End If
