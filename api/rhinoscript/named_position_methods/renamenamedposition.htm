RenameNamedPosition

Renames an existing named position.

Syntax

Rhino.RenameNamedPosition (strOldName, strNewName)

Parameters

strOldName

Required.  String.  The name of an existing named position.

strNewName

Required.  String.  The new name.

Returns

Boolean

True if successful, False otherwise.

Null

On error.

Example

Sub TestRenameNamedPosition

  Dim objPlugIn

  On Error Resume Next

  Set objPlugIn = Rhino.GetPluginObject("Named Position")

  If Err Then

    MsgBox Err.Description

    Exit Sub

  End If

  Call objPlugIn.RenameNamedPosition("MyNamedPosition", "YourNamedPosition")

End Sub
